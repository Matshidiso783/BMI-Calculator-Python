# BMI Calculator 

## Project Overview
A simple Python program that calculates Body Mass Index (BMI) based on user input.

##  How it works
- User enters weight (kg)
- User enters height (m)
- Output is classified into:
  - Underweight
  - Normal weight
  - Overweight
  - Obesity

##  Skills Used
- Python basics
- If/else statements
- User input handling
- Mathematical operations

## Purpose
To practice Python fundamentals and decision-making logic.
